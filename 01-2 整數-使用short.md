# 01-2 整數-使用short

## 1. 範例
``` java
//-----------------------
// 一個名稱為J01_2的類別
//-----------------------
public class J01_2{

    public static void main(String []args){
        
        // 建立一個short型態的整數變數i, 初值是32767
        short i = 32767;
        
        // 印出變數i的內容
        System.out.println(i);
    }
}
```
