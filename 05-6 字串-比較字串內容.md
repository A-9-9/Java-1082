# 05-6 字串-比較字串內容

## 1. 範例
``` java
//-----------------------
// 一個名稱為J05_6的類別
//-----------------------
public class J05_6{

    public static void main(String []args){
        
        // 設定學號
        String s = new String("10756001");        
        
        // 切割字串
        String year = s.substring(0, 3);
        System.out.println(year);
        
        // 判斷是否107學年度的學生	
        if (year == "107"){
        	System.out.println("是");
        }else {
        	System.out.println("不是");
        }  	
    }
}
```
