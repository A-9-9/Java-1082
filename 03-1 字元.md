# 03-1 字元

## 1. 範例
``` java
//-----------------------
// 一個名稱為J03_1的類別
//-----------------------
public class J03_1{

    public static void main(String []args){
        
        // 設定變數
        char a = 'A';
        char b = 67;        
        char c = 48;        
        char d = '林';
        
        // 印出變數值
        System.out.println(a);
        System.out.println(b);
        System.out.println(c);
        System.out.println(d);
    }
}
```
