# 01-6 整數-不同型別變數間的內容指派

## 1. 範例
``` java
//-----------------------
// 一個名稱為J01_6的類別
//-----------------------
public class J01_6{

    public static void main(String []args){
        
        // 建立2個變數
        int i = 10;
        short j = 20;
        
        // 印出變數內容
        System.out.println(i);
        System.out.println(j);   
        
        // 將變數j的值->變數i
        i = j;
        
        // 印出變數內容
        System.out.println(i);
        System.out.println(j);
    }
}
```
