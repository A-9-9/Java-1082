# 01-5 整數-相同型別變數間的內容指派

## 1. 範例
``` java
///-----------------------
// 一個名稱為J01_5的類別
//-----------------------
public class J01_5{

    public static void main(String []args){
        
        // 建立2個變數
        int i = 10;
        int j = 20;
        
        // 印出變數內容
        System.out.println(i);
        System.out.println(j);   
        
        // 將變數j的值->變數i
        i = j;
        
        // 印出變數內容
        System.out.println(i);
        System.out.println(j);
    }
}
```
