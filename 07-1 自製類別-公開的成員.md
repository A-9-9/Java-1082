# 07-1 自製類別-公開的成員

## 1. 範例
``` java
//-----------------------
// Student類別
//-----------------------
class Student{
    // 成員
    public String no;
    public String name;
}

//-----------------------
// 一個名稱為J07_1的類別
//-----------------------
public class J07_1{

    public static void main(String []args){
        
        // 產生Student類別的實體s
        Student s = new Student();
    	
        // 設定實體s的成員值
        s.no = "120001";
        s.name = "王小明";
    	
        // 印出實體s的成員
        System.out.println(s.no);
        System.out.println(s.name);   
    }
}
```
