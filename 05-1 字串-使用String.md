# 05-1 字串-使用String

## 1. 範例
``` java
//-----------------------
// 一個名稱為J05_1的類別
//-----------------------
public class J05_1{

    public static void main(String []args){
        
        // 設定變數
        String s1 = "Hello";
        String s2 = "王小明";
        
        // 印出變數值
        System.out.println(s1);
        System.out.println(s2);
    }
}
```
