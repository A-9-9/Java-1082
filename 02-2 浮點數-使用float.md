# 02-2 浮點數-使用float

## 1. 範例
``` java
//-----------------------
// 一個名稱為J02_2的類別
//-----------------------
public class J02_2{

    public static void main(String []args){
        
        // 建立變數
        float w = 10.5f;
        float h = 21;               
 
        // 計算
        float a = w * h;
        
        // 印出計算結果
        System.out.println(w);
        System.out.println(h);
        System.out.println(a);
    }
}
```
