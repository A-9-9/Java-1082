``` java

//-----------------------
``` 

## 3. 完成以下練習 (請用double型態儲存計算結果)

#### 3-1. 以下的程式在翻譯時出錯, 請修正它. (計算梯型面積: 上底是10.5, 下底是15, 高是18.5)
``` java
//--------------------------
// 一個類別, 名稱為J02_3a
//--------------------------
public class J02_3a{

    public static void main(String []args){
        
        // 計算
        float a = (10.5 + 15) * 18.5 / 2;
       
        // 印出計算結果
        System.out.println(a);
    }
}
```
