# 01-4 整數-使用long

## 1. 範例
``` java
//-----------------------
// 一個名稱為J01_4的類別
//-----------------------
public class J01_4{
 
    public static void main(String []args){
        
        // 建立一個long型態的整數變數i, 初值是12345
        long i = 12345;        
        
        // 印出變數i的內容
        System.out.println(i);
    }
}
```
