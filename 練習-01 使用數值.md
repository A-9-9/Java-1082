# 練習-01 使用數值

## 1. 電費-單一費率

#### (1) 說明

``` java
(1) 設定一個整數變數d, 內容存放[用電度數].

(2) 假設電費以[單一費率]收費, 收費標準如下, 請計算電費: 
    ...................................
                         每度（元）   
    ...................................
    120度(含)以下         2.10          
    121-330度             3.02        
    331-500度             4.39         
    501-700度             5.44        
    701-1000度            6.16         
    1001度以上            6.71         
    ...................................	  

    例如:用電750度, 其電費為:
         5.44 * 750  
         = 4080 (若有小數, 以四捨五入計)           
```

#### (2) 測試
``` java
註:
同學設定不同的用電度數後執行程式, 即可檢查自己計算的電費與以下參考答案是否相同?
同學的輸出可以不必用千分位數等格式顯示, 只要顯示數字即可.

(1) d = 100
    ==> 電費:210元
    
(2) d = 200
    ==> 電費:604元
    
(3) d = 400
    ==> 電費:1,756元
    
(4) d = 600
    ==> 電費:3,264元
    
(5) d = 800
    ==> 電費:4,928元
    
(6) d = 1200
    ==> 電費:8,052元
```


## 2. 電費-累計費率

#### (1) 說明

``` java
(1) 設定一個整數變數d, 內容存放[用電度數].

(2) 假設電費以[累進費率]收費, 收費標準如下, 請計算電費: 
    ...................................
                         每度（元）   
    ...................................
    120度(含)以下         2.10          
    121-330度             3.02        
    331-500度             4.39         
    501-700度             5.44        
    701-1000度            6.16         
    1001度以上            6.71         
    ...................................	  

    例如:用電750度, 其電費為:
          (2.10*120) + (3.02*210) + (4.39*170) + (5.44*200) + (6.16*50)
          = 3028.5
          = 3029  (小數四捨五入)       
```
