## 1. 電費-單一費率

#### (1) 說明

``` java
(1) 設定一個整數變數d, 內容存放[用電度數].

(2) 假設電費以[單一費率]收費, 收費標準如下, 請計算電費: 
    ...................................
                         每度（元）   
    ...................................
    120度(含)以下         2.10          
    121-330度             3.02        
    331-500度             4.39         
    501-700度             5.44        
    701-1000度            6.16         
    1001度以上            6.71         
    ...................................	  

    例如:用電750度, 其電費為:
         6.16 * 750  
         = 4620 (若有小數, 以四捨五入計)           
```

## 3. BMI-計算

#### (1) 說明

``` java
(1) 設定一個[身高]的變數h, [體重]的變數w.

(2) bmi = 體重 / (身高的平方)
          (其中, 體重以公斤為單位, 身高以公尺為單位)
          
    假設一個人的身高175公分, 體重78公斤.
    bmi = 78 / (1.75*1.75) = 25.47
```

#### (2) 測試
``` java
註: 以下的bmi值算至小數2位, 四捨五入

(1) h = 175
    w = 78
    ==> 25.47
    
(2) h = 163
    w = 50
    ==> 18.82
    
(3) h = 178
    w = 51
    ==> 16.10
    
(4) h = 163
    w = 77
    ==> 28.98
    
(5) h = 156
    w = 43
    ==> 17.67
```


## 4. BMI-評估

#### (1) 說明

``` java
(1) 設定一個[身高]的變數h, [體重]的變數w.

(2) bmi = 體重 / (身高的平方)
          (其中, 體重以公斤為單位, 身高以公尺為單位)
          
    假設一個人的身高175公分, 體重78公斤.
    bmi = 78 / (1.75*1.75) = 25.47
    
(3) bmi評估標準如下: 
    如果 bmi < 18.5, 評語是"過輕";
    如果 18.5 <= bmi < 24 , "正常";
    如果 24 <= bmi < 27, "過重";
    如果 bmi >= 27, "肥胖". 
```

#### (2) 測試
``` java
註: 以下的bmi值算至小數2位, 四捨五入

(1) h = 175
    w = 78
    ==> 25.47
    ==> 過重
    
(2) h = 163
    w = 50
    ==> 18.82
    ==> 正常
    
(3) h = 178
    w = 51
    ==> 16.10
    ==> 過輕
    
(4) h = 163
    w = 77
    ==> 28.98
    ==> 肥胖
    
(5) h = 156
    w = 43
    ==> 17.67
    ==> 過輕
```


## 5. 完美數

#### (1) 說明

``` java
(1) 如果將一個整數的所有因數相加(除了自己外), 其相加總和等於自己, 該數稱為完美數.   
    例如: 28的因數有1, 2, 4, 7, 14, 28. 除了自己外, 其他的因數總和 = 1 + 2 + 4 + 7 + 14 = 28, 
          所以28是一個完美數.
    
(2) 檢查一個數是否完美數.
```

#### (2) 測試
``` java
(1) 28
    ==> 是
    
(2) 496
    ==> 是
    
(3) 8128
    ==> 是
    
(4) 9386
    ==> 否
    
(5) 515
    ==> 否

(6) 24
    ==> 否
```


## 6. 繳款機

#### (1) 說明

``` java
(1) 設定[消費金額](price), [付款金額](pay) 等2個變數.

(2) 找錢金額 = 付款金額 - 消費金額
    (假設找錢金額>=0)
          
(3) 計算找錢金額應有幾個50元, 幾個10元, 幾個5元, 幾個1元.
```

#### (2) 測試
``` java
(1) 346, 500  
    ==> 50元:3個, 10元:0個, 5元:0個, 1元:4個
    
(2) 191, 200 
    ==> 50元:0個, 10元:0個, 5元:1個, 1元:4個
    
(3) 213, 1000 
    ==> 50元:15個, 10元:3個, 5元:1個, 1元:2個
    
(4) 415, 500 
    ==> 50元:1個, 10元:3個, 5元:1個, 1元:0個
```



## 7. 賓果

#### (1) 說明

``` java
(1)  建立 [投注號碼1](no1), [投注號碼1](no2), [投注號碼1](no3), [投注號碼1](no4) 等4個變數.
     代表一張彩卷投注的號碼, 它們不會重覆.
     
(2) 假設本期中獎號碼為: 14, 7, 10, 6. 開出的號碼也不會重覆.

(3) 比對彩卷對中幾個號碼.

(4) 假設對中4個號碼得100,000元; 
           3碼得15,000元; 
           2碼得2,000元;
```
