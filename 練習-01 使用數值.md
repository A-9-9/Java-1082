# 練習-01 使用數值

## 1. 電費-單一費率

#### (1) 說明

``` java
(1) 設定一個整數變數d, 內容存放[用電度數].

(2) 假設電費以[單一費率]收費, 收費標準如下, 請計算電費: 
    ...................................
                         每度（元）   
    ...................................
    120度(含)以下         2.10          
    121-330度             3.02        
    331-500度             4.39         
    501-700度             5.44        
    701-1000度            6.16         
    1001度以上            6.71         
    ...................................	  

    例如:用電750度, 其電費為:
         5.44 * 750  
         = 4080 (若有小數, 以四捨五入計)           
```

(1) d = 100
    ==> 電費:210元
    
(2) d = 200

(3) d = 400
    ==> 電費:1,756元

## 3. BMI-計算

#### (1) 說明

``` java
(1) 設定一個[身高]的變數h, [體重]的變數w.

(2) bmi = 體重 / (身高的平方)
          (其中, 體重以公斤為單位, 身高以公尺為單位)
          
    假設一個人的身高175公分, 體重78公斤.
    bmi = 78 / (1.75*1.75) = 25.47
```

#### (2) 測試
``` java
註: 以下的bmi值算至小數2位, 四捨五入

(1) h = 175
    w = 78
    ==> 25.47
    
(2) h = 163
    w = 50
    ==> 18.82
    
(3) h = 178
    w = 51
    ==> 16.10
    
(4) h = 163
    w = 77
    ==> 28.98
    
(5) h = 156
    w = 43
    ==> 17.67
```


## 4. BMI-評估

#### (1) 說明

``` java
(1) 設定一個[身高]的變數h, [體重]的變數w.

(2) bmi = 體重 / (身高的平方)
          (其中, 體重以公斤為單位, 身高以公尺為單位)
          
    假設一個人的身高175公分, 體重78公斤.
    bmi = 78 / (1.75*1.75) = 25.47
    
(3) bmi評估標準如下: 
    如果 bmi < 18.5, 評語是"過輕";
    如果 18.5 <= bmi < 24 , "正常";
    如果 24 <= bmi < 27, "過重";
    如果 bmi >= 27, "肥胖". 
```

#### (2) 測試
``` java
註: 以下的bmi值算至小數2位, 四捨五入

(1) h = 175
    w = 78
    ==> 25.47
    ==> 過重
    
(2) h = 163
    w = 50
    ==> 18.82
    ==> 正常
    
(3) h = 178
    w = 51
    ==> 16.10
    ==> 過輕
    
(4) h = 163
    w = 77
    ==> 28.98
    ==> 肥胖
    
(5) h = 156
    w = 43
    ==> 17.67
    ==> 過輕
```
