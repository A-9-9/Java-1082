# 01-7 整數-四則運算

## 1. 範例
``` java
//-----------------------
// 一個名稱為J01_7的類別
//-----------------------
public class J01_7{

    public static void main(String []args){
        
        // 建立2個變數
        int w = 10;
        int h = 20;     
               
        // 計算
        int a = w * h; 
        
        // 印出計算結果
        System.out.println(a);
    }
}
```
