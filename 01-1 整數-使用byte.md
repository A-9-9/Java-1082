# 01-1 整數-使用byte

## 1. 範例
``` java
import java.lang.*;

public class J01_1{

    public static void main(String []args){
        
        // 建立一個byte型態的整數變數i, 初值是100
        byte i = 100;
        
        // 印出變數i的內容
        System.out.println(i);
    }
}
```
