# 02-1 浮點數-使用double

## 1. 範例
``` java
//-----------------------
// 一個名稱為J02_1的類別
//-----------------------
public class J02_1{

    public static void main(String []args){
        
        // 建立變數
        double w = 10.5;
        double h = 21;               
 
        // 計算
        double a = w * h;
        
        // 印出計算結果
        System.out.println(w);
        System.out.println(h);
        System.out.println(a);
    }
}
```
