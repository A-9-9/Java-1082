# 05-2 字串-比對字串的地址

## 1. 範例
``` java
//-----------------------
// 一個名稱為J05_2的類別
//-----------------------
public class J05_2{

    public static void main(String []args){
        
        // 設定變數
        String s1 = "Hello";
        String s2 = "Hello";
        
        // s1和s2指向相同位址?
        System.out.println(s1 == s2);
        
        // s1和s2內容相同?
        System.out.println(s1.equals(s2));
    }
}
```
