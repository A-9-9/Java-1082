# 04-1 布林

## 1. 範例
``` java
//-----------------------
// 一個名稱為J04_1的類別
//-----------------------
public class J04_1{

    public static void main(String []args){
        
        // 設定變數
        boolean a = true;
        boolean b = false;
        boolean c = 1==10;
        boolean d = 1!=10;
        
        // 印出變數值
        System.out.println(a);
        System.out.println(b);
        System.out.println(c);
        System.out.println(d);
    }
}
```
